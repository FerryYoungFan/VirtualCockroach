------------------------------------简体中文------------------------------------
此 App 基于Adobe Air开发

对于Mac用户：
请下载 Air 包 （*.air文件）
无法直接打包 Adobe Air 运行环境给Mac，所以Mac系统需要去
https://get.adobe.com/cn/air/otherversions/
下载框架，版本选26.0及以上就可以。


对于Windows用户：
【如果您不信任单独的exe文件】
（从017b开始隐藏了初始窗口，可能杀毒软件误报）
可以使用zip文件，也可以使用此Air包来安装（*.air 文件）

请前往：
https://get.adobe.com/cn/air/otherversions/
下载框架，版本选26.0及以上就可以。


安装运行环境后就可以点击 VirtualCockroach V***.air 包来安装此软件了


---------------------------------------English---------------------------------------
This app was developed based on Adobe Air.

For Mac users:
Please download *.air pack
This application needs Adobe Air runtime for installing.
Please visit
http://www.adobe.com/go/getair/
and download the runtime for Mac (26.0+)



For Windows users:

*** If you do not trust the standalone *.exe file ***:
(From V017b, this app hides its initial window, which may be reported by antivirus software)
You can either use the zip file directly, or: You can install the Air pack (*.air file)

Please visit:
http://www.adobe.com/go/getair/
and download the runtime for Windows (26.0+)


Then you can install this application by clicking on  VirtualCockroach V***.air
